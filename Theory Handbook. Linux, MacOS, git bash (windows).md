
| Команда             | Что делает                            | Пример                 |
| ------------------- | ------------------------------------- | ---------------------- |
| `pwd`               | Показывает текущую папку              | `pwd`                  |
| `ls`                | Показывает файлы и папки              | `ls`                   |
| `ls -la`            | Показывает все файлы, включая скрытые | `ls -la`               |
| `cd`                | Переход в другую папку                | `cd Documents`         |
| `cd ..`             | Выйти на уровень выше                 | `cd ..`                |
| `mkdir`             | Создать папку                         | `mkdir project`        |
| `touch`             | Создать файл                          | `touch main.py`        |
| `cat`               | Показать содержимое файла             | `cat main.py`          |
| `nano`              | Открыть файл в редакторе nano         | `nano main.py`         |
| `cp`                | Скопировать файл или папку            | `cp file.txt copy.txt` |
| `mv`                | Переместить или переименовать файл    | `mv old.txt new.txt`   |
| `rm`                | Удалить файл                          | `rm file.txt`          |
| `rm -rf`            | Удалить папку                         | `rm -rf <folder>`      |
| `clear`             | Очистить терминал                     | `clear`                |
| `whoami`            | Показать текущего пользователя        | `whoami`               |
| `sudo`              | Выполнить команду от администратора   | `sudo apt update`      |
| `apt update`        | Обновить список пакетов               | `sudo apt update`      |
| `apt install`       | Установить программу                  | `sudo apt install git` |
| `python3 --version` | Проверить версию Python               | `python3 --version`    |
| `pip --version`     | Проверить версию pip                  | `pip --version`        |
| `git --version`     | Проверить версию Git                  | `git --version`        |
| `chmod`             | Изменить права файла                  | `chmod +x script.sh`   |
